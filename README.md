# Face Recognition

## Activate virtual environment
```
pipenv shell
```

## Install dependencies
```
pipenv install
```

## Identify persons
```
python3 identify-persons/identify-persons.py
```

## Run API for faces comparison
```
python3 compare-faces/api.py
```
